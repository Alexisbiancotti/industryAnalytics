2. Connection String Format:

postgresql+psycopg2://<username>:<password>@<host>:<port>/<database>
3. Components:

airflow: Username for connecting to the database.
airflow: Password for the specified user.
postgres: Hostname or IP address of the PostgreSQL server.

airflow: Name of the database to connect to.
